\# **Crop Production Prediction using Machine Learning**



&#x20;**Overview**

This project predicts crop production using Machine Learning based on factors like rainfall, temperature, and area.



&#x20;**Dataset**

The dataset includes:

\- State

\- Year

\- Crop

\- Season

\- Area

\- Rainfall

\- Temperature

\- Production



&#x20;**Technologies Used**

\- Python

\- Pandas

\- Scikit-learn

\- Matplotlib



&#x20;**Model Used**

\- Random Forest Regressor



&#x20;**Results**

The model predicts crop production with good accuracy using environmental factors.
